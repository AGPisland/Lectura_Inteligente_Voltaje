ESTADOS POSIBLES:
ESTANQUE VACIO SIN IMPORTAR EL NIVEL,
	DIA:
		MUCHO SOL:
				BATERIAS BAJAS:
						NIVEL BULK:
									IMPLICA QUE EL NIVEL DE CORRIENTE ES ALTO DEBIDO AL NIVEL SOLAR, Y LA BATERIA SE ENCUENTRA EN AUMENTO DE CARGA.
									ESTE NO ES UN ESTADO OPTIMO PERO ENCAMINA HACIA EL OPTIMO.
						
				BATERIAS ALTAS:
						NIVEL ABSORPTION:
									IMPLICA UN NIVEL DE CORRIENTE MENOR AL MAYOR POSIBLE Y MAYOR AL MENOR POSIBLE.
									ESTE ES EL ESTADO MEDIO OPTIMO DEL SISTEMA.
						NIVEL MAINTENANCE:
									IMPLICA UN NIVEL DE CORRIENTE MUY MENOR.
									ESTE ES EL ESTADO MAS OPTIMO DEL SISTEMA.
		POCO SOL:
				BATERIAS BAJAS:
						NIVEL BULK:
									EL NIVEL DE CORRIENTE DEBE SER BAJO, EL SISTEMA NO TIENE ENERGIA PARA PODER CARGAR LAS BATERIAS DE TODAS FORMAS
									SE CARGAN PERO EL TIEMPO REQUERIDO SERA MAYOR.

				BATERIAS ALTAS:
						NIVEL ABSORPTION:
									EL NIVEL DE CORRIENTE TAMBIEN ES BAJO PERO DEBE COMENZAR EN DECLIVE, NO ES CLARO SABER REALMENTE SI EL SISTEMA
									ESTA EN EL NIVEL ABSORPTION O NIVEL MAINTENANCE DEBIDO A QUE LA FALTA DE ENERGIA SOLAR PRODUCE UN BAJO NIVEL DE 										CORRIENTE.
						NIVEL MAINTENANCE:
									EL NIVEL DE BATERIA DEBE SER EL MISMO NIVEL DE MAITENANCE DEL ESTADO DE BATERIAS ALTAS CON EL NIVEL DE SOL ALTO.
									EL SISTEMA DEMORO MUCHO EN LLEGAR A ESTE ESTADO DEBIDO A QUE EL NIVEL DE ENERGIA BAJO SOLAR IMPLICA UNA CARGA SUPER
									LENTA.
